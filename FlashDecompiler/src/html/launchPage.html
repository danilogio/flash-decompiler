Flash Decompiler Launch Page


Welcome to our project - a flex decompiler from within flex. The dropdown box enables you to pick your method of retrieving a swf to parse.


Once you parse your swf, you may save it for easy access later.


This project is underway so visit the project's home page for more information.


To make it easy, we've included a swf (this projects's own code) for you to decompile. A link to it should be added below, so just click the parse button to get started!


Also, if you are retrieving swfs via the file browser or database drowser, note that you must double click on them to parse. This way they won't behave weirdly under ordinary circumstances.
